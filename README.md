# Chaotic Binger's Longest Streak

This project determines the longest streak of consecutive episodes watched in order, given a list of watched episodes.

## Files
- ChaoticBinger.java : Java solution
- test_cases.txt : List of test cases
- output.txt : Output of the program

## How to Run
Compile and run ChaoticBinger.java using any Java IDE or javac/java commands.
